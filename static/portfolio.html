This page contains projects that I've been working on as a student at Ada Developers Academy.

Solo Projects

Solar System

Solar System was our first project working with Ruby classes, inheritance and writing methods. This simple little terminal program gives the user information about a planet in the Solar System based on their choice. For me, this was the first time I felt like I truly understood classes and how to use them.

I'm, like, overly proud of the user experience stuff I added to this project. It made it really fun to work on.

Check out the code at my GitHub Repository: Solar System!

Bank Accounts

Bank Accounts was another Ruby project working with classes, but this time we were importing data from csv files. This project really helped me solidify writing methods (because we had to write a LOT of them!).

Check out the code at my GitHub Repository: Bank Accounts!

FarMar

FarMar (an abbreviation for Farmer's Market) was our first solo Ruby project working with spec testing. Having worked on a pair project that used spec testing, I didn't feel like I quite had the hang of it. This project really solidified testing for me.

The project itself contains many different Ruby classes and methods, as well as many different CSV files of data. Understanding how each of these files were talking to each other was the goal of this project, and I feel like I really got it.

I also had a breakthrough on removing redundancy in my code, and spent an important work day revising and restructuring. I think my end result was better for it.

Check out the code at my GitHub Repository: FarMar

Pair Programming Projects

Word Guess

Word Guess was our first project done in pair programming. Written in Ruby, we made a "hangman style" word guessing game, where a user would guess letters and the program would give them progress on how they were doing--via ASCII art!

This project was challenging, because it was done so early in my Ruby educational journey, but ultimately I'm pleased with what we were able to do--even with our limited knowledge.

Check out the code at my GitHub Repository: Word Guess

Scrabble
