Ruby Madlib

As part of the Ada application process, aspirants worked in this problem-solving where we had to make a Madlib Game in Ruby.

It was my first approach to Ruby, and a very easy way to understand basic concepts of the program: puts, local variable, user input, strings, intepolation, encatatition, and comments.

Puts vs. Print

puts is an amazing tool while learning. It allows you to see what is going on. It prints out in your console-terminal, messages por the user to read. Later in my learning I found out that puts is very usful for debugging, specially through loops.
