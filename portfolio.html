Solar System

Learning Milestones

  • Object Oriented Programming
  • Classes & Composition
  • Variables & Scope

Solar System was an early project I worked on alone to learn about classes and object oriented programming. This was the first project where we were instructed to write our own classes. We also had to use composition to use the Planet class inside of the Solar System class. The program was to be execute in the terminal and involved user interaction to learn about different planets. I had fun writing a 'Space ID' random name generator and a secret easter egg in the program.

Code Sample


  # Generates a random number with user selected amount of digits
  # doesn't account for digits <= 0
  def random_number(digits)
    if digits == 1
      return rand(0..9)
    else
      range_start = 10 ** (digits - 1)
      range_end = (10 ** (digits)) - 1
      return rand(range_start..range_end)
    end
  end
  # Generates a random letter from a to z
  def random_letter
    return ("a".."z").to_a[rand(26)]
  end
  # Generate a random space id for user
  def random_space_id
    "#{random_number(1)}#{random_letter.upcase}" +
    "#{random_number(3)}-#{random_number(2)}" +
    "#{random_letter.upcase}"
  end

Word Guess Game

Learning Milestones

  • Pair Programming
  • Classes
  • Git
  • Gems

Word Guess Game was our first pair programming assignment. Much of our time was spent learning to work with another person on a project, emphasizing communication and pair programming techniques, as well as using git with another person. We were also still learning core concepts including classes and methods. One thing I'm proud of for this assignment is our awesome dragon ASCII art which used the colorize gem to really impress the player.
